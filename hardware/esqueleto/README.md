# Esqueleto

Subpasta para componentes estruturais do braco robotico.

## Objetivo mecanico

- Garantir rigidez estrutural para reduzir vibracao.
- Garantir alinhamento entre eixo do motor, redutor e junta.
- Permitir manutencao e troca de componentes.
